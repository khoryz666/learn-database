Reference variables
❑Directly reference a specific database field or record 
and assume the data type of the associated field or 
record
❑%TYPE: same data type as a database field
❑%ROWTYPE: same data type as a database record
◼To declare a variable named current_f_last that references 
values retrieved from F_LAST in FACULTY table
❑Current_f_last FACULTY.F_LAST%TYPE
◼To declare a variable that references an entire row of data in 
FACULTY table
❑Faculty_row FACULTY%ROWTYPE

Manipulating Character Strings with PL/SQL
◼To concatenate two strings in PL/SQL, you use the 
double bar (||) operator:
❑new_string := string1 || string2 ;
◼To remove blank leading spaces use the LTRIM 
function:
❑string := LTRIM(string_variable_name);
◼To remove blank trailing spaces use the RTRIM 
function:
❑string := RTRIM(string_variable_name);
◼To find the number of characters in a character string use 
the LENGTH function:
❑string_length := LENGTH(string_variable_name);
22 |-UCCD2303/UCCD2203 -|
Manipulating Character Strings with PL/SQL
◼To change case , use UPPER, LOWER, INITCAP
◼INSTR function searches for a specific substring 
and return an integer that represent the starting 
position of the substring.
❑start_position := INSTR(original_string, substring);
◼SUBSTR function extracts a specific number of 
characters from a character string, starting at a 
given point:
❑extracted_string := SUBSTR(string_variable,  
starting_point, number_of_characters);
