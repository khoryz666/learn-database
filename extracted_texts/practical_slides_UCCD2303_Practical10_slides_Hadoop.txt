Definition of Big Data

Definition of Big Data
|-UCCD2303/UCCD2203 -| 4
◼Big Data is a collection of data that is huge in 
volume yet growing exponentially with time. 
◼It is a data with very large size and 
complexity that none of traditional data 
management tools can store it or process it 
efficiently. 
◼Big data is also a data but with huge size.
Definition of Big Data
|-UCCD2303/UCCD2203 -| 5
◼Following are some examples for Big Data:
❑The New York Stock Exchange is an example of 
Big Data that generates about one terabyte of 
new trade data per day.
❑The statistic shows that 500+ terabytes of new 
data get ingested into the databases of social 
media site Facebook , every day. 
◼This data is mainly generated in terms of photo and 
video uploads, message exchanges, putting comments 
etc.
Definition of Big Data
|-UCCD2303/UCCD2203 -| 6
◼Following are some examples for Big Data 
(cont.):
❑A single Jet engine can generate 10+ terabytes 
of data in 30 minutes of flight time. With many 
thousand flights per day, generation of data 
reaches up to many Petabytes .
Note: 
More details on Big Data and Hadoop will be covered during the 
Lecture class.

Listing, Inserting, and Retrieving Data 
from Hadoop File System
|-UCCD2303/UCCD2203 -| 39
◼Hadoop Distributed File System (HDFS) is the 
primary data storage system used by Hadoop 
applications.
❑It is about how Hadoop organizes all the files which 
were assigned to it.
◼HDFS is similar to Linux File System except for 
the following:
❑HDFS does not remember its current directory/path –
it is called ‘stateless’, unlike Linux.
❑Each time has to specify the entire directory/path in 
HDFS.
